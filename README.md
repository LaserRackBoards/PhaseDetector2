# PhaseDetector2
Phase detector with wide bandwidth, for PDH lock

* SYPD-1, SYPD-2, SYPD-52W phase detector
* [ADC-10-1R](https://www.minicircuits.com/pdfs/ADC-10-1R) directional coupler, 10.5dB coupling, 0.7dB mainline loss
* [BUF602](https://www.ti.com/product/BUF602) buffer amp
* [ADA4898-1](https://www.analog.com/en/products/ada4898-1.html) low noise op-amp
* [LT1568](https://www.analog.com/en/products/lt1568.html) low noise active filter
